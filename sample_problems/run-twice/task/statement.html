配点 : 11 点

問題文

0, 1, …, 9 からなる数字列 S が与えられます。
S を並び替えて得られる数字列であって、十進数として解釈したとき素数になるようなものは何種類ありますか？

制約

  • S は 0, 1, …, 9 からなる文字列である。
  • 1 ≤ |S| ≤ 11 (|S| は S の長さを表す)

入力

入力は以下の形式で標準入力から与えられる。

S

出力

答えを出力せよ。

入力例 1

139

出力例 1

2

139 を並び替えて得られる数字列は 139, 193, 319, 391, 913, 931 の 6 種類ありますが、このうち十進数として解釈したとき素数になるようなものは 139, 193 の 2 種類です。

入力例 2

110

出力例 2

2

110 を並び替えて得られる数字列は 011, 101, 110 の 3 種類ありますが、このうち十進数として解釈したとき素数になるようなものは 011, 101 の 2 種類です。

入力例 3

01123456789

出力例 3

1404250
